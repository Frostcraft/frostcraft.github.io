About us

We are a gaming community that has been around since 2012. Primarily we've hosted a Minecraft server, but we may occasionally try other games too. Some different things we've tried over the years have been Minecraft (factions, feed the beast, custom magic server), and recently Vintage story.

Vintage Story Server

Vintage story is a mostly un-modified server except for a discord chat integration. You can check the game out at https://www.vintagestory.at/


Server address: vs.frostcraft.com

Minecraft Magic Server

The Minecraft magic server is driven by a custom plugin. On the server are classes, professions, magic spells, and much more.


Server address: mc.frostcraft.com
